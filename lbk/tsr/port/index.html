Sobre o Projeto

Área com protocolo para transformação em ZEIS, conforme o programa ProZEIS da prefeitura municipal de Maringá, com previsão de aprovação no início de 2025.

O estudo protocolado prevê a construção de aproximadamente 1.500 apartamentos, viabilizando edifícios T+6 no padrão do programa Minha Casa Minha Vida (MCMV).

As infraestruturas externas não incidentes no empreendimento (como vias públicas, drenagem, etc.) serão custeadas pelo proprietário, que é dono de uma loteadora. Assim, as áreas serão entregues "líquidas" para construção.
